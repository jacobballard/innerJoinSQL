INNER JOIN

SQL INNER JOIN works pretty simply on our end. It joins data from one table with that of another table based on
simple parameters. The code is as follows:

  1. "SELECT (column_1), (column_2)" The columns selected will the columns that return information based on your later parameters.
  2. "FROM (original table)" Choose a table that holds your common identification (i.e. OrderID).
  3. "INNER JOIN (table)" Choose a table that you wish to join this information with.
  4. "ON (original_table_column = other_table_column)" This is the conditional that allows you to determine which datasets will be joined.
    If the data does not match, the data will not be joined.
